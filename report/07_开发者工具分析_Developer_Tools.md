## Cursor：从零到王者

Cursor 于 **2024-07** 首次进入 Top 100，此后从未缺席。

- 首月流量：31,682
- 最新月流量：732,969
- 增长倍数：**23.1x**
- 最高份额：5.8%

Cursor 的成功不是偶然的。它的增长曲线几乎是教科书式的 SaaS 增长：

1. **产品力驱动**：AI 代码补全 + 内置聊天 + VS Code 兼容性，降低了迁移成本
2. **开发者的付费意愿高**：对于每天使用 8 小时的工具，0/月的定价完全可以接受
3. **网络效应**：团队中一人使用后，往往会带动整个团队付费
4. **持续迭代**：几乎每周都有产品更新，保持了用户黏性

### Lovable & Bolt.new：AI 建站赛道

与 Cursor 侧重专业开发者不同，Lovable 和 Bolt.new 瞄准的是"非程序员也能做应用"的市场。

**lovable.dev**：2024-12 首次出现，最新流量 830,594，出现 15 个月

**bolt.new**：2024-10 首次出现，最新流量 35,926，出现 17 个月

### Replit & Devin：差异化路线

Replit 走的是云端 IDE + AI 路线，Devin 则是全自动 AI 软件工程师。两者在 Top 100 中出现的月份有限，说明这两个方向的付费转化尚未成熟。

## 洞察：为什么开发者工具是 Stripe 的黄金赛道

1. **高 ARPU**：开发者工具的月订阅价格通常在 0-50/月，远高于消费类应用
2. **低流失率**：一旦形成使用习惯，开发者很少切换工具（迁移成本高）
3. **B2B 扩展**：个人使用后推动团队采购，从 B2C 自然过渡到 B2B
4. **全球市场**：编程工具天然全球化，不受地域限制

从 Stripe 的视角看，AI 编程工具可能是其最优质的客户群体之一——高频、高价、低流失。

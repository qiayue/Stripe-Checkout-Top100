## 数据说明

每月数据来源于 SimilarWeb 对 checkout.stripe.com 的入站流量分析，记录了每月向 Stripe 支付页面贡献最多流量的 Top 100 网站。每条记录包含域名、访问量、流量份额、行业类别等信息。

## 核心发现速览

- **Stripe 支付生态稳健增长**：月流量从 14.1M 增长至 19.3M，26 个月增长 37%
- **AI 重塑付费格局**：AI 产品在 Top 100 中的流量占比从约 10% 飙升至约 27%，成为最大变量
- **王朝更替**：Roblox 统治 22 个月后让出王座，此后 Suno、Manus、Grok 轮流称王，再无霸主
- **开发者工具崛起**：Cursor 从 0 到稳居 Top 3，AI 编程工具成为新的付费主力
- **市场走向分散**：Top 5 集中度显著下降，长尾效应增强，Stripe 生态更加健康
